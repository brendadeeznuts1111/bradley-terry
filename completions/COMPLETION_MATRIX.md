## Bunfig settings (non-CLI)

Canonical `bunfig.toml` keys live in `completions/bunfig-settings.json` (18 settings).

Examples that are **not** duplicated as global CLI flags:
- `jsx` — JSX transform mode (also in tsconfig compilerOptions)
- `logLevel` — Log verbosity for Bun runtime
- `telemetry` — Enable/disable anonymous crash reports
- `test.preload` — Preload scripts for `bun test` only
- `test.smol` — `smol` mode for `bun test` only
- `test.coverage` — Enable test coverage collection
